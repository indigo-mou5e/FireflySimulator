# FireflySimulator
蛍の発光シミュレーターです。  
それぞれの蛍が周りの蛍の発光状況を参照し、光るタイミングを変えます。  
その変え方の規則(アルゴリズム)をいくつか搭載しているので、どのアルゴリズムならうまく同調するか調べることが出来ます。  
観賞用としても使えます。
